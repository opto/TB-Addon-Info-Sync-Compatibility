• Addons for search, message display, (auto)archiving, moving/copying messages, adding notes, image display:


    • Expression Search NG: Enhance search capabilities of (quick)search: f:Fred t:John a:yes searches for all messages from Fred to John with an attachment (and many other search enhancements) Link to ATN

    Quicker Quicksearch: easily search for recipients/domain/subject of highlighted email in quicksearch. Link to ATN

    Nostalgy++: easy archiving/moving of emails to their destination folders, with autocomplete of folder names and rules for suggested folder. Suggestion of folders for move/copy by statistical observation of previous move operations. Multiple folder search: "in no" finds 'Inbox/Thunderbird/addons/Nostalgy' folder with typing of 4 characters, only. Compose window: always display cc or bcc, easy selecting input fields (to, cc, bcc, body, attachments), etc. Link to ATN

    Text insert: Easy inserting standard text. br+ctrl-D converts to best regards. Define your own replacements. After insert, the cursor can be in the middle of the replacement text, to continue writing between a 'hello' and the regards. Link to ATN

    Attachment image viewer: Image viewer with zoom, rotate, flip, slideshow etc. Attachment download dialog shows thumbnail of image. Link to ATN

    New mail to all: Like a reply to all, but subject, body and attachments are empty. This is an easy way to start a new email to a set of recipients that were part of a previous conversation. This command starts a new conversation. Link to ATN

    XNote++: Add notes to your emails to remind or to plan a reply. Link to ATN
