Features:

Easily insert standard text into email composer

  • Please prepare a list of shortcuts and replacements in the addon options (tools -> addons)/b>

  • Typing shortcut followed by ctrl-d replaces the shortcut

    • Example: br(ctrl-d) expands to best regards

  • Replacements may contain line breaks

  • Edit old shortcuts/replacements: just click into the cell to edit. On removing focus, the new text is accepted. Save the table.

  • Setting cursor position: if the replacement text holds an #c, the cursor will be set to that position

  • Known issue: if returning to the table, newlines are no longer shown, but they are still applied in composer
