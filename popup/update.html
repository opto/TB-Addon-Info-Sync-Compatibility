Features


Addon-Info/Sync/Compatibility is an extension to Thunderbird which gives info on installed addons and on their compatibility with the next ESR version, TB 91.

Example:
  • Display addon info, addon version, addon max version (published on ATN) and addon compatibility in the addon table
  • Print table of addons
  • Copy list of addons to clipboard
  • Sync/Email list of addons to yourself. On receiving, another TB will be able to highlight not installed addons, if {addon} is installed.

You can report issues at the home page for TB78: Addon-Info/Sync/Compatibility. It is gratefully appreciated if you can support the transfer of my addons to TB 91 by a donation.
